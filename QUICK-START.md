# 🚀 异国小助手 - 快速启动指南

## 📋 问题解决

你遇到的PowerShell错误已经修复！PowerShell不支持`&&`操作符，我已经创建了多个启动方式。

## 🎯 推荐启动方式

### 方式1: 双击启动（最简单）
```bash
# 双击运行这个文件
quick-start.bat
```

### 方式2: PowerShell脚本
```powershell
# 在PowerShell中运行
.\start.ps1
```

### 方式3: 手动启动
```powershell
# 1. 启动后端（在一个窗口）
node server.js

# 2. 启动前端（在另一个窗口）
cd client
npm run dev
```

## 🌐 访问地址

启动成功后，在浏览器中访问：
- **前端界面**: http://localhost:3001
- **后端API**: http://localhost:3000

## ⚙️ 邮件配置（重要！）

1. 复制环境变量文件：
```bash
copy .env.example .env
```

2. 编辑 `.env` 文件，配置邮箱：
```env
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password
```

**注意**:
- 使用Gmail需要开启两步验证并生成应用专用密码
- 不要在代码中直接写入密码

## 🔧 常见问题

### Q: 启动时端口被占用？
```bash
# 查看占用端口的进程
netstat -ano | findstr :3000
netstat -ano | findstr :3001

# 结束进程
taskkill /PID <进程ID> /F
```

### Q: 邮件发送失败？
- 检查 `.env` 文件配置
- 确认邮箱密码是应用专用密码
- 检查网络连接

### Q: 前端无法连接后端？
- 确认后端服务器正在运行
- 检查端口是否正确（3000）
- 查看浏览器控制台错误信息

## 📞 技术支持

如果还有问题：
1. 检查Node.js版本：`node --version`（需要16+）
2. 清理npm缓存：`npm cache clean --force`
3. 重新安装依赖：删除node_modules后重新npm install
